## [1.0.0] — YYYY-MM-DD

### Added — Initial Release
- **🎮 Video Game Discovery**: RAWG API integration for game search
- **📰 Gaming News**: Real-time news feed via external API
- **👥 Social Platform**: Friend management (add, remove, search)
- **👤 User Profiles**: Editable profiles with avatar support
- **❤️ Personal Library**: Game history and favorites tracking
- **🔐 Authentication**: Firebase Auth + Google Sign-In + Guest mode
- **🌙 Dark/Light Theme**: Material Design 3 with DataStore persistence

### Technical Foundation
- MVVM + Clean Architecture with Jetpack Compose
- Hilt dependency injection
- Room local database + Firestore cloud sync
- Navigation Compose type-safe routing
- Retrofit + OkHttp networking layer
